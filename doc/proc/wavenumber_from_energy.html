public function wavenumber_from_energy(energy_) result(k_)

returns the wavenumber, \(k_{a}\), given the energy of a given state, \(E_{a}\); calls etot() function; atomic units in the whole function \( k_{a} = \sqrt(2 \mu (E_{tot} - E_{a}) \) since it uses reducedmass and etotal(), the function checks if units are already converted

Arguments

Type IntentOptional Attributes Name
real(kind=dp), intent(in) :: energy_

energy of a given state, \( E_{a} \), in a.u.

Return Value real(kind=dp)

wavenumber, \(k_{a}\), in a.u.
