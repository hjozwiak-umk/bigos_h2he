This module contains subroutines that transform the asymptotic log-derivative matrix into the scattering S-matrix: 1. pfunc - calculates the P coefficients from Eq. (6.19) 2. transsum - calculates the sum over \bar{\Omega} in Eq. (6.17-6.18) 3. bftosfmat - transforms a given BF-matrix to the SF frame of rerence 4. calculate_k_matrix - transforms log-derivative matrix into reactance matrix

  1. kmattosmat - transfroms reactance matrix into scattering S-matrix
