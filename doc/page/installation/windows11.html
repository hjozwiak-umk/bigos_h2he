Before building the code, make sure that we have all the necessary compilers, librariries and apps installed:

Here, we provide a step-by-step instruction for installation of the necessary packages using Cygwin.

  1. Download the Cygwin installer from the official website

  1. Launch Cygwin setup.
  2. Choose "Install from Internet".
  3. Set up a Root directory.
  4. Set up a local package directory.
  5. Choose "Use System Proxy Settings".
  6. Choose A Download Site.
  7. After a while, a new "Select Packages" window appears. Navigate to "Full" list in the upper right corner:

  • Locate necessery packages: make, gcc-fortran, liblapack-devel, libopenblas, git, wget and unzip. Choose the most recent version.

  • Proceed with the installation and close the Setup window.

  • Launch Cygwin Terminal

  • Clone the repository from git git clone https://github.com/hjozwiak-umk/bigos_h2he.git ./bigos_dir

  • Enter ./bigos_dir

  • Run make all.

Note

The script will download and extract the wigxjpf library in ./libs directory,
and then build and compile the contents of ./src directory.
The executable, scattering.x, is located in the main directory.

You can now run make test to make sure that you recover reference outputs in the ref/ directory.
